CSS ビヘイビアプロパティ

2007-12-17
memo

behaviorプロパティはIE独自の拡張機能なのですが、コレを利用するとIEだけできない諸問題を解決できそうです。
pngファイルのサポートや:hoverのaタグ以外への適用、min-widthなどの問題などなど

利用方法はどうやらHTC（HTML Component）を作成し、HTML上で使用する。
HTCファイルはjavascriptなどで構成されていて、特定のイベント時などに実行されて、
ソースコードが書き換えられる。
W3C邦訳

behaviorプロパティはIE5.0以降に実装されているので、対象ブラウザとしてIE5.xを含むときにこの方法でさまざまな諸問題を回避できるのでは無いでしょうか。。。

と、web creatorsを読んでいて思った。。。
