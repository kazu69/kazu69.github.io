YUIのfont.cssとオペラの文字サイズ

2007-12-02
memo

YUIのfont.cssにてデフォルトCSSを作成して、利用しているのですが、どうもOperaでは、フォントサイズを100%以下にすると、フォントがほかのブラウザより、小さく表示されるバグがあるようです。

で、対処方法の一案がありました。

【解決編】Yahoo! UI Library: Fonts CSSを使うのなら、Operaには気を付けて！ - TRANS [hatena]を参考に。

どうも、YUIの値を若干ずらすということらしい。

YUIも不完全ということでしょうか。。。とりあえず、使えそうな気がします。
