Sass、Less以外のCSS PreprocessorとGUIのコンパイルアプリをまとめてみた

2012-06-11
memo

SassとLess以外にもCSS Preprocessorはいろいろあるので、メモがわりにまとめてみた。

  • Stylus
    node.jsのCSS Preprocessor
  • TASS
    Javascriptでできた軽量CSS Preprocessor
  • Turbine
    Sass記法っぽいPHPのCSS Preprocessor
  • PCSS
    PHP5で使えるCSS Preprocessor
  • CSS Crush
    高機能なPHPベースのCSS Preprocessor
  • DtCSS
    CSSの記法とほぼ同じなCSS Preprocessor
  • CSS CACHEER
    Apache(mod_deflate,mod_rewrite)とPHPが必要なCSS Preprocessor
  • less.java
    LessのJava版Parser
  • lesscss-java
    おなじくLessのJava版Parser
  • pyScss
    PythonのScssコンパイラー
  • lesscss-python
    PythonのLessコンパイラー

ざっくり調べてもけっこうあった。
コンパイルできるアプリも、

  • Scout
    Mac、Winで使える無料のSassコンパイルできるアプリ
  • LiveReload
    Mac、Winで使えるCoffeeScript、SASS、LESSコンパイルできるアプリ
  • Codekit
    Macで使えるLess, Sass, Stylus, Jade, Haml, CoffeeScriptコンパイルできるアプリ。画像も最適化できる。
  • Compass.app
    Mac、Winで使えるSass,Compass Frameworkをコンパイルできるアプリ
  • Less.app
    Macで使えるLessをコンパイルアプリ
    うーん、やっぱりCodekitが一番高機能だな。フロント周りの主なPreprocessorはほぼ使える。
