2014-01-05
node

HarpはMarkdown, Jade, EJS, LESS, Stylus, Sass, CoffeeScriptを内蔵したウェブサーバー。

プリプロセッサー内蔵しているので、CoffeeScriptなどをコンパイルすることなく公開できる。

Harpは、node.jsでのMiddlemanやJekyllのような静的なサイトのジェネレーターのような感じ。
