モジュールとしてファイルをimport

v4.0ではimport周りが大幅に変わるらしい。

  1. @import_once の追加
  2. cssファイルのimportが可能に
  3. 名前空間の追加
  4. 自動でindex.scssまたはindex.sassを検索するようになる
  5. folder単位でのimportパスが使用可能になる
    と大幅に変更されいままでいろいろと面倒だったか点を解決できそう。
