1PasswordがWindowsに対応

2010-05-09
web

macとiphoneのパスワード管理でいい感じ、というか最強と思っている1PasswordがWindowsに対応してくれました。

ちなみに、1Passwordというのは、パスワードを一元で管理できるツール。一度入力したパスワードを管理し、次回からのパスワード入力を自動化してくれます。現在はMac、iphone対応のアプリケーション提供しています。

今回リリースされた1Password for Windowはまだbeta版(公式リリースまでは無料)ですが、今のところ充分利用はできる感じです。
ブラウザとしてはIE7、Firefox3に対応という事で今後はChromeにも対応予定とか。これは嬉しいです。

Windowsで利用する前に、パスワードのデータをDropboxなどで同期しておくこともお忘れなく。
