容量の大きなファイルを簡単に作る方法

2009-06-24
memo

ほら、仕事とかで負荷テストで容量の大きめなファイルとか必要になるじゃないですか。

あれを簡単に作る方法を、忘れないようにメモしておく。

コマンドプロンプトで　sutil file createnew ファイル名 サイズすればいいらしい。

たとえば、Cドライブ直下に１GBのファイルを作るなら、

C:>fsutil file createnew testfile 10737418240

でできる。

これで、負荷テストも簡単にできそう。

[参考:http://www.atmarkit.co.jp/fwin2k/win2ktips/243largefile/largefile.html]
